2.2 假设 API 模式的成本估算

Claude Haiku 定价:

  • 输入：$0.80/100K tokens

  • 输出：$4.00/100K tokens

    • 单次发言成本:
    `
    预估输入 tokens = ~700 tokens
    • System prompt：~200 (人格 + 技能 + 背景)
    • User prompt：~500 (话题 + 上下文)

    预估输出 tokens = ~150 tokens

    成本 = (700/100K × $0.80) + (150/100K × $4.00)
    = $0.0056 + $0.0006
    = $0.0062 / 次发言
    `

    如果满额运行（默认配置）:

  • 参与人数：20 Agent

  • 话题数：4 个

  • 轮数：2 轮/话题

  • 发言人数：5 人/轮

  • 总发言数 = 20 × 4 × 2 × 5 = 800 次

    • `
    满额成本估算 (API 模式):
    • 总输入 = 800 × 700 = 560K tokens
    • 总输出 = 800 × 150 = 120K tokens
    • 输入成本 = (560K/100K) × $0.80 = $4.48
    • 输出成本 = (120K/100K) × $4.00 = $4.80
    • 总成本 = $9.28
